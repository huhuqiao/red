第三方商家入驻流程

注册

注册 小红书商家入驻系统，用注册邮箱和收到的原始密码，登录小红书商家入驻系统。

填写入驻信息

填写公司信息、财务信息、店铺信息等，并提交相应的资质。

资质审核

小红书将尽快反馈审核结果。

合同签署

您可在 线上确认合同内容。审核通过后，我们会给您寄纸质合同。您在 线下签署合同 后，需将 合同寄回。

完成培训

完成小红书商家培训后，你就可以登录 小红书商家管理平台开店了！

常见问题

流程问题 资质问题 收费与结算 管理后台功能

Q：入驻小红书的流程大概需要多长时间？

A：入驻的时间主要取决于商家本身：如果商家可以快速填写入驻信息，并一次性通过审核，且不对合同进行申诉，那么最快可以一周内完成入驻。

Q：如何修改入驻平台的密码？

A：入驻平台暂不支持修改密码。请保存好邮件密码，以便重复使用。

Q：如何查询入驻进度？

A：回到入驻平台查看，可获得最准确的进度信息。

Q：怎么知道审核是否通过？

A：小红书回复后，入驻平台会立即更新，登录入驻平台可获得最准确的进度信息。同时，系统会发送邮件至“店铺管理人邮箱”进行提醒。

Q：如果已在小红书开通过店铺，能否使用卖家入驻系统再开通新店铺？

A：可以。小红书实行“单店铺单账号”体系，您可以使用多个账号开设多家店铺。

Q：如何选择想要开设的店铺类型？系统会帮我决定吗？

A：首先，商家在入驻流程中选择想要开设单品牌店铺，或多品牌店铺。如果您是品牌商以及品牌授权的商家，您必须开设“单品牌店铺”，如果您拥有多个品牌，您需要开设多个店铺；如果您是贸易商、零售商等，您必须选择“多品牌店铺”。小红书暂不接受只销售单品牌的贸易商和零售商。之后，系统会通过您填写的信息，以及提交的资质类型，按官方规则自动匹配店铺类型。请保证您填写的信息准确且真实，如果提交后不能通过审核，您需要重新修改，可能耽误入驻进度。

Q：签署合同的流程是怎样的？

A：在入驻流程中确定了合同内容之后，还需要在线下完成签署：小红书的法务会将[盖好公章的一式两份合同]寄往商家提供的经营地址。商家务必在收到合同的2个工作日内寄回。如果在法务寄出后的1个月内，商家尚未寄回，小红书可能会冻结商家账号。

Q：如果Q&A无法解答问题，应该如何寻求帮助？

A：可以通过 ft_sellerservice@xiaohongshu.com 联系我们的客服。但为了不耽误您的入驻，希望您可以先仔细阅读本网站上的信息再联系客服。

Q：三证合一是什么意思？

A：所谓“三证合一”，就是将企业依次申请的营业执照、组织机构代码证和税务登记证三证合为一证。如果您已经拥有三证合一证，可直接上传，如果还没有，那么需分别上传三证。

Q：填写银行打款信息时是否需要上传开户证明？如果是，应该选择哪一种？

A：需要。请询问您的财务，了解您公司账户是“一般户”还是“基本户”。如果是“一般户”，请上传“开立单位银行结算账户申请书”。如果是“基本户”，请上传“银行开户许可证”。请注意，填写的账户信息一定要是贵司的“企业银行账户”，而非个人银行账户。

Q：需要上传的资质有哪些？

A：入驻平台一共需要上传三类资质：公司资质、品牌资质、品类资质。详见“入驻前”页的信息和资质准备。

Q：小红书如何收费？

A：销售佣金是唯一的收费模式；没有其他任何营销、流量和资源位费用。小红书会和申请入驻的商家在入驻流程的“合同签署”步骤确定佣金率。

Q：小红书如何结账？

A：每一个自然月为一个结算周期，以结算周期内小红书系统记录的入驻店铺已发货数据为准。每月月初，商家可以在后台管理系统查收并审核“账单管理”。（如商家按时在系统里审核账单且无异议）小红书应于每月15日向商家支付上一个自然月的销售款，遇节假日顺延。

Q：小红书的平台保证金是多少？应该如何支付？

A：不同的商家有不同的保证金，保证金根据商家特点、资质类型和店铺类型而变化。小红书会和申请入驻的商家在入驻流程的“合同签署”步骤确定保证金。小红书保证金,需要商家通过公对公转账到小红书账户。

Q：商家和小红书之间的发票如何处理？

A：小红书向商家开具与上一个自然月收取佣金等额的6%增值税专用发票。

Q：需要完成哪些任务，商品才能自动上架？

A：完成以下几个步骤即可（具体信息可以在通过资质审核后，加入官方培训群了解）： 1. 您需要在入驻流程中，完成对合同内容的确认，并走完入驻流程 2. 登入管理后台，进入“商家设置”tab，完成“基本信息”和“售后信息”的填写 3. 登入管理后台，进入“搜索/添加商品”tab上传商品： a.输入spv，spl，spl_item, item等信息，并通过审核 b.提供商品图片、商品名、商品特色等必填内容 c.填写价格，完成税率的校验 d.如果您进行的是海外贸易，还需在上传商品时提供海关备案信息

Q：如何使用小红书后台？商品上传有什么要求？

A：商家通过第二步的资质审核后，会需要进入培训群接受培训和测试。培训材料中有对小红书后台使用方法的详细信息。

Q：物流模式怎么确定？商家需不需要入小红书仓？

A：物流模式和是否需要入小红书仓都是自选的（合同中会有物流内容，但合同的签署不视为要求商家使用小红书提供的物流方案）。在商家成功入驻，并进入管理后台后，可以在“物流模式”tab完成所有物流信息的填写和配置：可以选购小红书的物流解决方案，也可选择自行发货（具体信息可以在通过资质审核后，加入官方培训群了解）

Q：是否可以通过对接 API 对接订单和库存？

A：小红书提供 API 接口和文档，供商家自行对接订单/库存模块。目前小红书已经和国内多个主要的ERP系统实现了订单和库存对接。开店入驻不以 API 对接为前提。如果不对接，可以在小红书卖家后台进行手动维护。

Q：小红书的退换货政策是什么？

A：发货地在境内：7天无理由退货；发货地在境外：暂时无要求。
